CFG Sentence Generator

Enter a valid CFG below, and click "generate" to create a random sentence.

  • The first production must be SENTENCE
  • Each production should be of the form: NAME -> prod1 | prod2 | ...
  • Use the EPSILON production to represent nothing

To assign probabilities for each production being chosen, add them as a ~ followed by a comma-seperated list of numbers to represent the probability of each e.g. ~ 1,7,3,4. See the expression example for this in action.
